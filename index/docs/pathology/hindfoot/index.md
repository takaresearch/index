# Hindfoot Pathology

後足部の主要な病態について、頻度順に整理しています。

## Common Conditions (by Frequency)

1. **Ankle Osteoarthritis** - Most common degenerative condition
2. **Osteochondral Defect (OCD)** - Common in young athletes
3. **Achilles Tendinopathy** - Frequent overuse injury
4. **Ankle Instability** - Common after ankle sprains

## Anatomical Considerations

The hindfoot includes:
- Ankle joint (tibiotalar joint)
- Subtalar joint
- Achilles tendon complex
