# Forefoot Pathology

前足部の主要な病態について、頻度順に整理しています。

## Common Conditions (by Frequency)

1. **Hallux Valgus** - Most common forefoot deformity
2. **Hallux Rigidus** - Common degenerative condition
3. **Morton's Neuroma** - Frequent cause of forefoot pain
4. **Metatarsalgia** - General forefoot pain syndrome

## Anatomical Considerations

The forefoot includes:
- First metatarsophalangeal (MTP) joint
- Lesser MTP joints
- Intermetatarsal spaces
- Toes
