# Midfoot Pathology

中足部の主要な病態について、頻度順に整理しています。

## Common Conditions (by Frequency)

1. **Flatfoot (PTTD)** - Most common acquired flatfoot deformity
2. **Lisfranc Injury** - Important traumatic condition

## Anatomical Considerations

The midfoot includes:
- Tarsometatarsal (Lisfranc) joints
- Naviculocuneiform joints
- Posterior tibial tendon

Each pathology in this section covers:
