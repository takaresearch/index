## 1. 研究の背景と実施の意義・必要性

### 1-1. 研究の背景

骨粗鬆症は、骨強度の低下を特徴とし、骨折のリスクを増大させる全身性骨格疾患です。世界保健機関(WHO)によると、50歳以上の女性の3人に1人、男性の5人に1人が、生涯のうちに骨粗鬆症性骨折を経験すると推定されています。日本においても高齢化の進展に伴い、骨粗鬆症患者は約1,280万人と推定され、その約80%が未診断・未治療の状態にあるとされています。

骨粗鬆症性骨折、特に椎体骨折と大腿骨近位部骨折は、患者の生活の質(QOL)を著しく低下させ、医療経済的負担も大きくなっています。早期発見と適切な介入により、骨折を予防し、健康寿命を延伸することが可能です。

現在、骨粗鬆症診断のゴールドスタンダードは、二重エネルギーX線吸収測定法(DXA法)による骨密度測定です。しかし、DXA装置は高額であり(1台約1,000万円以上)、医療施設への設置が限られているため、特に地方や発展途上国においてスクリーニングのアクセシビリティが課題となっています。また、放射線被曝の問題(微量ではあるが)や、測定に要する時間とコストも、大規模スクリーニングの障壁となっています。

### 1-2. 姿勢と骨密度の関連性

胸椎後弯(脊柱の前後方向への湾曲の増大、いわゆる「猫背」)と骨密度低下の関連性は、多くの疫学研究で報告されています。Ensrudら(1997)は、Framingham研究のコホートを用いて、後弯角度の増大が低骨量と関連することを示しました(J Am Geriatr Soc. 1997;45(6):682-687)。Kadoら(2009)のStudy of Osteoporotic Fracturesでは、後弯角度の増大が独立した骨折リスク因子であり、さらに死亡率の上昇とも関連することが示されました(Ann Intern Med. 2009;150(10):681-687)。

重要なことに、Schneiderら(2004)の研究では、無症候性椎体骨折を持たない女性においても、後弯角度と腰椎骨密度の間に有意な負の相関が認められました(J Rheumatol. 2004;31(4):747-752)。これは、後弯が単に骨折の結果ではなく、骨質の低下を反映する独立したマーカーである可能性を示唆しています。

### 1-3. コンピュータビジョンと機械学習の医療応用

近年、深層学習技術の急速な発展により、画像からの人体姿勢推定の精度が飛躍的に向上しています。OpenPose(Cao et al., IEEE Trans Pattern Anal Mach Intell. 2021)やMediaPipe(Bazarevsky et al., 2020)などの最先端アルゴリズムは、2次元画像から高精度に身体の主要な関節点(キーポイント)を検出できます。

医療分野においても、内視鏡画像診断、網膜疾患診断、皮膚癌診断など、画像診断と機械学習の組み合わせによる成功例が多数報告されています。整形外科領域でも、脳性麻痺患者の歩行解析(Kidziński et al., Nat Commun. 2020)や脊柱側弯症のスクリーニング(Aroeira et al., J Bodyw Mov Ther. 2016)への応用が報告されています。

しかし、骨粗鬆症スクリーニングへの姿勢推定技術の応用は、まだ初期段階にあります。

### 1-4. 研究の意義・必要性

本研究は、深層学習ベースの姿勢推定アルゴリズムを用いて、通常のカメラで撮影した立位側面写真から抽出した姿勢パラメータと、年齢・性別を組み合わせて骨密度を予測する機械学習モデルの精度を検証することを目的としています。

本研究の実施により、以下の意義が期待されます：

1. **医療アクセスの向上**: スマートフォンやタブレットのカメラで撮影可能な写真から骨密度を推定できれば、DXA装置へのアクセスが限られている地域や、移動困難な高齢者に対しても、スクリーニングの機会を提供できます。

2. **医療経済的効果**: 低コストで実施可能なスクリーニングにより、骨粗鬆症の早期発見率が向上し、結果として高額な骨折治療費(大腿骨近位部骨折1件あたり約200万円)を削減できる可能性があります。

3. **非侵襲性**: 放射線被曝がないため、繰り返しの評価や若年者への適用も可能です。

4. **業務効率化**: 第一段階スクリーニングとして使用することで、DXA検査を真に必要とする高リスク者を効率的に選別でき、医療資源の適正配分に貢献します。

5. **科学的貢献**: 姿勢推定技術の骨粗鬆症スクリーニングへの応用は、国際的にも先駆的な試みであり、学術的価値が高いと考えられます。

本研究は予備的・探索的研究として位置づけられ、将来的な大規模検証研究や実用化への重要な基盤となります。

## 2. 研究の目的

### 2-1. 主要目的

深層学習ベースの姿勢推定アルゴリズム(MediaPipe PoseまたはOpenPose)を用いて立位側面写真から抽出した姿勢パラメータ(胸椎後弯角度、頭部前方位、腰椎前弯角度など)と、年齢・性別を組み合わせた機械学習モデルによる骨密度予測の精度を検証する。

### 2-2. 副次目的

1. 姿勢パラメータとDXA法で測定した骨密度(腰椎および大腿骨近位部)の相関を明らかにする
2. 年齢・性別のみのベースラインモデルと比較して、姿勢パラメータを含むモデルが骨密度予測精度を有意に改善するかを検証する
3. 異なる機械学習アルゴリズム(線形回帰、ランダムフォレスト、勾配ブースティング、ニューラルネットワークなど)の性能を比較する
4. 骨粗鬆症(T-score ≤ -2.5)の分類精度を評価する

### 2-3. 研究仮説

- **仮説1**: 胸椎後弯角度は骨密度と有意な負の相関を示す(相関係数 r < -0.4, p < 0.05)
- **仮説2**: 姿勢パラメータを含むモデルは、年齢・性別のみのモデルと比較して、骨密度予測の決定係数(R²)を少なくとも0.1改善する
- **仮説3**: 機械学習モデルによる骨粗鬆症分類のAUC(Area Under the Curve)は0.75以上を達成する

## 3. 研究対象者（対象患者）

### 3-1. 選択基準（適格基準）

以下の全ての基準を満たす者を研究対象者とする：

1. 年齢: 40歳以上80歳以下(同意取得時)
2. 独立して立位保持が5分以上可能な者
3. 本研究の内容を理解し、書面によるインフォームドコンセントを提供できる者
4. 研究参加に同意した者

### 3-2. 除外基準

以下のいずれかに該当する者は研究対象者から除外する：

1. 骨粗鬆症治療薬(ビスホスフォネート製剤、デノスマブ、テリパラチド、SERM、抗RANKL抗体、ロモソズマブなど)の現在または過去の使用歴がある者
2. 脊椎または下肢の手術歴がある者
3. 脊柱変形(脊柱側弯症Cobb角>20度、強直性脊椎炎など)の診断を受けている者
4. 神経筋疾患(パーキンソン病、筋ジストロフィー、脳血管障害後遺症など)により姿勢制御に重大な障害がある者
5. 妊娠中または妊娠の可能性がある女性
6. DXA測定が禁忌の者(体重制限超過、金属インプラントにより正確な測定ができない者など)
7. 研究参加に同意しない者、または同意能力がないと判断される者
8. その他、研究責任者または分担研究者が研究対象者として不適切と判断した者

### 3-3. 目標症例数と設定根拠

**目標症例数: 50名**

#### 設定根拠:

先行研究(Ensrud et al., 1997; Schneider et al., 2004)において、後弯角度と骨密度の相関係数は約r = -0.45と報告されています。効果量r = 0.4、有意水準α = 0.05(両側検定)、検出力1-β = 0.80とした場合、必要なサンプルサイズは46名です(Cohen, 1988)。

多変量回帰分析においては、予測変数1つあたり最低10症例が推奨されています(Harrell et al., 1996)。本研究では約8-10個の予測変数を用いるため、50名のサンプルサイズは適切です。

脱落や測定エラーを考慮し、目標症例数を50名とします。

### 5-3. データ収集方法

#### 5-3-1. 写真撮影

**撮影環境の標準化:**
- 撮影場所: 屋内の明るい部屋、または均一な照明下
- 背景: 無地の白または明るい色の壁
- カメラ高さ: 被験者の大転子の高さに固定(三脚使用)
- 撮影距離: 被験者から2.5メートル
- カメラ: デジタルカメラまたはスマートフォンカメラ(解像度1920×1080ピクセル以上)

**被験者の準備:**
- 服装: 身体のラインが識別可能な薄手の衣服(Tシャツとレギンスなど)
- 履物: 裸足または薄いソックス
- 髪型: 長髪の場合は束ねて頸部・背部のラインが見えるようにする

**撮影手順:**
1. 被験者に自然な立位姿勢を取るよう指示
2. 完全な側面(矢状面)から撮影(右側面)
3. 「普段の楽な姿勢で立ってください」と指示し、5秒間保持後に1枚目を撮影
4. 一度歩いてもらい、再度立位をとってもらい2枚目を撮影
5. 全身(頭頂から足底まで)が画面内に収まるよう確認
6. 顔は写らないように配慮(首から下を撮影、または顔にモザイク処理)

**所要時間**: 約10分

#### 5-3-2. 骨密度測定(DXA法)

**測定装置**: [DXA装置のメーカー・モデル名]

**測定部位**: 
- 腰椎(L1-L4)
- 大腿骨近位部(全体、頸部、転子部)

**測定者**: 認定技師が実施

**測定項目**:
- 骨密度(BMD): g/cm²
- T-score: 若年成人平均値との標準偏差比較
- Z-score: 同年齢平均値との標準偏差比較

**放射線被曝**: 約0.001 mSv(通常の胸部X線検査の約1/10)

**所要時間**: 約20分

**測定タイミング**: 写真撮影と同日、または1週間以内

#### 5-3-3. 質問紙調査

以下の項目について、研究対象者に質問紙に回答してもらいます：

**基本情報:**
- 年齢、性別、身長、体重、BMI

**既往歴・家族歴:**
- 骨折歴(部位、年齢)
- 骨粗鬆症の家族歴
- 現在治療中の疾患
- 手術歴

**薬剤使用歴:**
- ステロイド使用歴(3ヶ月以上)
- カルシウム・ビタミンDサプリメントの使用

**女性の場合:**
- 初経年齢、閉経年齢
- 妊娠・出産回数

**生活習慣:**
- 喫煙歴(現在喫煙、過去喫煙、非喫煙)
- 飲酒習慣(頻度と量)
- 運動習慣(頻度と内容)
- 食習慣(牛乳・乳製品、魚、大豆製品の摂取頻度)

**所要時間**: 約10分

### 5-4. 実施手順・方法

#### ステップ1: スクリーニングと同意取得

1. 研究対象候補者を選定し、適格基準・除外基準を確認
2. 研究の説明を行い、十分な考慮期間を提供
3. 書面によるインフォームドコンセントを取得
4. 研究対象者としてエントリー

#### ステップ2: データ収集

1. 質問紙への回答(約10分)
2. 身長・体重の測定
3. 立位側面写真の撮影(2枚、約10分)
4. DXA法による骨密度測定(約20分)

**合計所要時間**: 約1時間

#### ステップ3: データ管理と匿名化

1. 収集したデータ(写真、DXA結果、質問紙)を研究用データベースに入力
2. 各研究対象者に一意の研究ID番号を割り当て
3. 個人識別情報を削除し、匿名化
4. 匿名化されたデータを解析用データセットとして保存
5. 対応表(研究ID-個人情報)は、研究責任者が厳重に管理

#### ステップ4: 姿勢推定とパラメータ抽出

1. 姿勢推定アルゴリズム(MediaPipe PoseまたはOpenPose)を用いて、写真から身体のキーポイントを自動検出
2. 検出されたキーポイントから、以下の姿勢パラメータを算出:
   - 胸椎後弯角度(Thoracic Kyphosis Angle)
   - 頭部前方位(Forward Head Posture)
   - 腰椎前弯角度(Lumbar Lordosis Angle)
   - 矢状面垂直軸(Sagittal Vertical Axis, SVA)
   - 体幹傾斜角度(Trunk Inclination)
   - 頭頸部角度(Craniovertebral Angle)
3. 2枚の写真から得られた値の平均を使用
4. 測定の信頼性(級内相関係数ICC)を評価

#### ステップ5: 統計解析

**5-5-1. 記述統計:**
- 研究対象者の特性(年齢、性別、BMI、骨密度、姿勢パラメータなど)の記述統計を算出
- 骨密度分類(正常、骨減少症、骨粗鬆症)ごとの特性を比較

**5-5-2. 相関分析:**
- 姿勢パラメータと骨密度(腰椎BMD、大腿骨頸部BMD)の相関を評価
- Pearsonの相関係数またはSpearmanの順位相関係数を算出
- 多重比較補正(Bonferroni補正)を実施

**5-5-3. 機械学習モデルの構築:**

以下の機械学習アルゴリズムを用いて、骨密度予測モデルを構築:
1. 線形回帰(Linear Regression)
2. リッジ回帰(Ridge Regression)
3. ランダムフォレスト(Random Forest)
4. 勾配ブースティング(XGBoost)
5. サポートベクター回帰(SVR)
6. ニューラルネットワーク(Multi-Layer Perceptron)

**特徴量:**
- ベースラインモデル: 年齢、性別
- 拡張モデル: 年齢、性別、姿勢パラメータ(6個)、BMI

**モデル評価:**
- 5分割交差検証(5-fold cross-validation)
- 評価指標: 決定係数(R²)、平均絶対誤差(MAE)、二乗平均平方根誤差(RMSE)

**5-5-4. 分類性能評価(骨粗鬆症 vs 非骨粗鬆症):**
- ROC曲線とAUC(Area Under the Curve)
- 感度、特異度、陽性的中率、陰性的中率
- 最適カットオフ値の決定(Youden指数)

**5-5-5. 多変量解析:**
- 重回帰分析: 骨密度を従属変数、姿勢パラメータ・年齢・性別を独立変数とする
- 標準化偏回帰係数、95%信頼区間、p値を算出

**統計ソフトウェア:**
- Python 3.8以上(NumPy, Pandas, Scikit-learn, SciPy, Statsmodels, TensorFlow/PyTorch)
- 有意水準: α = 0.05(両側検定)

## 7. 研究対象者に生じる負担並びに予測されるリスク及び利益

### 7-1. 予測される利益

#### 直接的利益:
1. **無料での骨密度測定**: 通常は自費で数千円かかるDXA検査を無料で受けられます。
2. **早期発見**: 骨粗鬆症が発見された場合、早期に治療を開始できます。
3. **専門医への紹介**: 必要に応じて、適切な医療機関を紹介します。
4. **健康意識の向上**: 自身の骨の健康状態を知ることで、予防行動の動機付けになります。

#### 間接的利益:
1. **社会貢献**: 本研究の成果は、将来的に多くの人の骨粗鬆症予防に役立つ可能性があります。
2. **医療の発展**: 非侵襲的スクリーニング法の開発に貢献します。

### 7-2. 予測されるリスク・不利益

#### 写真撮影に関して:
- **心理的負担**: 身体の写真を撮影されることに抵抗を感じる可能性があります。
  - **対策**: プライバシーに十分配慮し、研究者以外は入室しません。顔は写らないよう配慮します。

#### 骨密度測定(DXA)に関して:
- **放射線被曝**: ごくわずかな放射線被曝があります(約0.001 mSv、胸部X線検査の約1/10)。
  - **対策**: 被曝量は極めて微量であり、健康への影響は無視できるレベルです。妊娠中または妊娠の可能性がある女性は除外します。
- **所要時間**: 測定に約20分かかります。
- **心理的負担**: 骨密度が低いことがわかり、不安を感じる可能性があります。
  - **対策**: 必要に応じて、適切な医療機関をご紹介し、専門医による説明とフォローアップを受けられるようにします。

#### その他:
- **時間的負担**: 来院とデータ収集に合計約1時間かかります。
- **交通費**: 来院のための交通費は自己負担となります。

### 7-3. リスクの最小化

1. 経験豊富な研究者が、標準化されたプロトコルに従ってデータ収集を行います。
2. 研究対象者のプライバシーに最大限配慮します。
3. 異常所見が見つかった場合、速やかに適切な対応を取ります。
4. 研究参加中、いつでも質問や相談ができる体制を整えます。
