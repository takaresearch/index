# 運用・公開ポリシー（必読）

本サイトは、足部・足関節外科に関する知識と作業手順を、Markdownで継続的に整理するための公開ノートである。一方で、医療情報は本質的に再同定リスクを含み、公開に伴う倫理的・法的責任が不可避である。したがって、本ポリシーは「何を載せるか」ではなく、まず「何を載せないか」を明示し、公開可能性の境界を固定することを目的とする。

## 1. 公開してよい情報（原則）

- **一般化された知識**: 診断アルゴリズム、鑑別の軸、評価手順、術式選択の前提、合併症と回避策など。  
- **教育的テンプレート**: 外来プレゼンの型、問診の型、手術記録の型（具体症例に依存しない骨格）。  
- **既発表の業績**: 公開済み論文・発表・研究費等の書誌情報。  

## 2. 公開してはいけない情報（禁止）

- **症例同定性のある情報**（単体または組合せで特定されうるもの）  
  - 日時、施設、地域、稀少疾患、特徴的既往、職業、競技歴、きわめて特異な画像所見の連続提示等  
- **未匿名化データ**  
  - 画像DICOM、生データ、検査結果、自由記載、音声、動画等  
- **未公開の研究内容**  
  - 投稿準備中原稿、査読対応、解析途中の結果、共同研究の未公表情報  

## 3. 「研究：執筆中」を確実に非公開にする運用

本リポジトリでは、執筆中原稿を `private/` 以下に置く。`private/` は `.gitignore` により追跡されず、GitHub Pages の公開物にも含まれない。  
ただし、これは「公開リポジトリに載らない」ための技術的措置であり、機微情報の保護を完全に代替するものではない。機密性の高い情報は、所属施設の規程および情報セキュリティ方針に従うこと。

## 4. 公開前チェック（最小チェックリスト）

- 症例同定性のある要素が残っていないか（組合せによる再同定を含む）  
- 画像・図表の権利（撮影者・施設・出版社）の問題がないか  
- 共同研究・学会発表の未公表情報が混入していないか  
- 「次回アクション」「術中分岐」等の記載が、個別症例の時系列や希少性を露呈していないか
