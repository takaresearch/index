# 研究

本ページ群では、研究活動を「既発表（公開）」と「執筆中（非公開）」に厳密に分離して管理する。  
GitHub Pages に掲載されるのは **既発表の業績（公開可能な情報）** のみであり、執筆中原稿は公開リポジトリの追跡対象外として運用する。

## 非公開（執筆中）の取り扱い

執筆中の原稿・メモ・解析途中の結果は、リポジトリ直下の `private/` に配置することを推奨する。`private/` は `.gitignore` により追跡されず、GitHub Pages にも含まれない。  
ただし、**機密性の高いデータ（症例情報、未匿名化データ等）** は、本仕組みの範囲外として、所属施設の規程と情報セキュリティ方針に従って管理する必要がある。
