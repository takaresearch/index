# 臨床

本ページ群では、足部・足関節領域の臨床トピックを、**臨床推論の手順が追跡できる形**で整理する。具体的には、(1)鑑別診断の軸、(2)評価手順（身体所見・画像・検査）、(3)治療選択（保存／手術）と適応、(4)合併症と再手術シナリオ、を最小単位として記載する。

## 運用方針

- **公開範囲**: 教育的・一般化可能な内容に限定し、症例同定性のある情報は扱わない。  
- **更新**: トピック単位で追記し、後日修正を前提とする（“生きたノート”）。  

## これから追加する項目（例）

- 外反母趾：術式選択の前提（変形要素の分解）
- 後脛骨筋腱機能不全：ステージングと治療戦略
- 足関節不安定症：保存療法の限界と術式選択
- 距骨軟骨損傷：病態分類と治療アルゴリズム
