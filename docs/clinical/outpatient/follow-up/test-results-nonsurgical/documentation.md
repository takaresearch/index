## d. Recommendation

### Template

**Plan**:

1. **Continue/Modify Conservative Management**:
   - Physiotherapy: [Continue current program / modify to focus on [ASPECT]]
   - Orthotics: [Continue / adjust / trial different type]
   - Medication: [Continue / taper / add [AGENT]]
   - Activity: [Progress activity / maintain restrictions]
   - Additional modality: [Consider injection / shockwave / other]

2. **Timeline**:
   - Trial period: [DURATION]
   - Reassessment goal: [specific improvement target]

3. **Follow-up**:
   - Review in [TIMEFRAME]
   - Repeat imaging: [if indicated - WHEN and WHY]

4. **Decision Points**:
   - **If improved**: 
     - Wean support gradually
     - Progress to functional activities
     - Discharge or PRN follow-up
   
   - **If no improvement or deterioration**:
     - Consider [injection therapy / surgical consultation]
     - Reassess diagnosis
     - Additional investigations if indicated

**Patient Education**:
- Expected timeline for improvement: [TIMEFRAME]
- Signs of progress to monitor: [specific markers]
- Red flags: [when to seek urgent review]

**Correspondence**: Follow-up letter to GP / referring clinician with results and plan
