## d. Recommendation

### Template

**Plan**:

1. **Investigations**:
   - [X-ray / MRI / CT / Ultrasound] [LOCATION]
   - Purpose: to assess [bone stock / ligament integrity / tendon pathology]
   - Expected timeframe: [~4-6 weeks]

2. **Conservative Management** (trial for [DURATION]):
   - Physiotherapy: [specific focus - strengthening, stretching, proprioception]
   - Orthotics: [heel cups, insoles, AFO as appropriate]
   - Medication: [NSAIDs / simple analgesia] as needed
   - Activity modification: [specific advice]
   - Immobilization: [CAM boot / splint] if indicated

3. **Follow-up**:
   - Review in [~3 months / after investigations complete]
   - Goals: reassess symptoms, review imaging, determine next steps

4. **Decision Points**:
   - **If improved**: Continue conservative measures, wean support, return PRN
   - **If no improvement**: Consider [injection / surgical consultation / alternative diagnosis]
   - **If deterioration**: Urgent review, modify treatment

**Patient Education**:
- Expected recovery timeline: [TIMEFRAME]
- Activity restrictions: [specific guidance]
- Red flags: [infection signs, neurovascular compromise, worsening pain]

**Correspondence**: Letter to GP / referring clinician
