    Follow-up: Post-Op Subacute - Documentation¶

    a. History Taking¶

    Template¶

    Postoperative Time: [WEEKS/MONTHS] post [PROCEDURE], performed [DATE]

    Functional Status: - Walking distance: [DISTANCE] without significant pain - Stairs: [managing independently / requires rail / difficulty] - Work status: [returned full duties / modified duties / not yet returned] - Sport status: [returned / progressing / not yet]

    Symptoms: - Pain: [VAS, description - at rest / with activity / specific triggers] - Swelling: [resolved / minimal after activity / persistent] - Stiffness: [resolved / improving / persistent - worse in morning / after rest]

    Rehabilitation: - Physiotherapy: [number of sessions completed, ongoing/discharged] - Home exercise compliance: [good / moderate / poor] - Specific progress: [strength, ROM, proprioception improvements noted]

    Patient Satisfaction: [very satisfied / satisfied / concerns / dissatisfied]

    Complications: [none / specific issue]

    b. Examination¶

    Template¶

    Gait: [normal / antalgic / uses walking aid]

    Inspection: - Scar: [well-healed / hypertrophic / keloid / other] - Swelling: [absent / minimal / moderate / significant] - Alignment: [maintained / concerns]

    Palpation: - Tenderness: [resolved / minimal / persistent over LOCATION] - Hardware: [not prominent / palpable / prominent / painful]

    Range of Motion: - [JOINT]: Active [measurements] / Passive [measurements] - Compared to contralateral: [equal / 75% / 50% / <50%] - End feel: [normal / stiff / painful]

    Strength: - [Muscle groups tested]: [5/5 / 4/5 / 3/5 compared to contralateral]

    Functional Tests: - Single leg stance: [able TIME seconds / unable] - Heel/toe walking: [able / unable] - [Other relevant functional tests]

    Imaging: - X-ray: [hardware position stable / alignment maintained / fusion progressing / implant stable] - [Other imaging if indicated]

    c. Impression¶

    Template¶

    Postoperative Course: [WEEKS/MONTHS] post [PROCEDURE]

    Overall Progress: [Excellent / Good / Satisfactory / Below expected] functional recovery

    Specific Findings: - ROM: [recovered to X% of contralateral / excellent / limited] - Strength: [recovered / improving / residual weakness] - Function: [returned to LEVEL of activity]

    Imaging: [Hardware/fusion/repair] [stable and well-positioned / healing appropriately / concerns]

    Complications: [None / resolved / ongoing issue]

    Assessment: Patient with [excellent / good / satisfactory] functional outcome following [PROCEDURE]. [Achieving goals / on track / requiring continued support].

    d. Recommendation¶

    Template¶

    Plan:

    1. Activity:
    2. Progress to [NEXT LEVEL: unrestricted activities / return to sport / full duties]
    3. Timeline: [gradual over TIMEFRAME / immediate / specific milestones]

    4. Restrictions: [none / avoid high-impact for DURATION / specific limitations]

    5. Rehabilitation:

    6. Continue [home exercise program / discharged from PT / ongoing PT]
    7. Focus: [maintenance / final strength gains / sport-specific training]

    8. Duration: [ongoing / until MILESTONE]

    9. Return to Activities:

    10. Work: [full clearance / specific restrictions / timeline]
    11. Driving: [cleared / restrictions]

    12. Sport: [cleared / gradual return / specific timeline]

    13. Imaging:

    14. [Repeat X-ray at TIMEFRAME if indicated / no further imaging needed]

    15. Follow-up:

    16. If progressing well:

      • Final review in [TIMEFRAME]
      • Then discharge to GP care

    17. If concerns persist:

      • Review in [SHORTER TIMEFRAME]
      • [Additional investigations / interventions as needed]

    18. Open door: Contact if new problems arise

    19. Long-term Management:

    20. Expected full recovery: [TIMEFRAME]
    21. Continue exercises for [DURATION] for optimal outcome
    22. Annual X-rays for [TYPE OF SURGERY] recommended
    23. OR: No long-term follow-up needed unless symptomatic

    Discharge Planning (if appropriate): - Patient discharged from foot & ankle service - Satisfactory outcome achieved - GP to manage any minor musculoskeletal issues - Re-referral pathway available if needed

    Patient Education: - Expectations for final outcome - Timeline for residual improvements - When to seek review - Maintenance exercises for long-term outcome

    Correspondence: Discharge summary to GP / final follow-up letter
