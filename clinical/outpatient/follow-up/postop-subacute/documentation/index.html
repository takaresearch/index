    Follow-up: Post-Op Subacute - Documentation¶

    Letter Structure¶

    This template follows the standard outpatient letter format: Diagnosis → Plan → Main paragraph → Impression and recommendation.

    For subacute postoperative follow-up (typically 6 weeks to 6 months), focus on healing progression, functional recovery, return to activities, and discharge planning.

    1. Diagnosis¶

    Format: Time post-op + Procedure + Current recovery status

    Template: - [TIME: ~6 weeks / ~3 months / ~6 months] post [SPECIFIC PROCEDURE], performed [DATE] - [Recovery status descriptor]

    Common descriptors: - progressing well | satisfactory recovery | excellent functional outcome - good radiographic healing | solid fusion | bone healing progressing - achieving functional goals | returned to most activities

    Example:

    Diagnosis
- ~6 months post right ankle arthroscopy and lateral ligament reconstruction, performed 15/07/25
- Excellent functional outcome with achievement of all goals

    2. Plan¶

    Format: Activity progression → Rehabilitation status → Discharge vs continued follow-up → Long-term considerations

    Template:

    Activity Progression: - Current activity level: [describe what patient can do now] - Cleared for: [specific activities - full activity / return to sport / return to work] - Progression guidance: [gradual increase / specific restrictions / sport-specific advice] - Timeframe: [when to expect full function]

    Rehabilitation Status: - Physiotherapy: [completed / ongoing / discharged from physio] - Home exercise program: [continue / can stop / maintenance program] - Specific goals achieved: [ROM / strength / proprioception milestones] - Further therapy needed: [yes - for DURATION and GOALS / no further formal therapy needed]

    Return to Activities:

    Work: - Status: [returned to full duties / modified duties / not yet returned] - Clearance: [cleared for full duties / continue modifications for DURATION / gradual return advised] - Restrictions: [none / avoid SPECIFIC ACTIVITY for DURATION]

    Driving: - Status: [already driving / cleared to resume] - Guidance: [specific advice if relevant - e.g., right ankle surgery and emergency braking]

    Sport: - Status: [returned / progressing toward return / not yet] - Clearance: [cleared for all sports / specific sports / contact sports at TIMEFRAME] - Progression: [gradual return over DURATION / sport-specific program / return when comfortable]

    Follow-up Plan:

    If Progressing Well: - Discharge: To GP care with open-door policy - Annual review: If indicated (e.g., for hardware, arthroplasty, fusion assessment) - PRN return: If concerns, symptoms worsen, or new problems develop

    If Concerns or Ongoing Issues: - Continued follow-up: Review in [TIMEFRAME] - Goals for next visit: [assess specific concern / finalize recovery / hardware removal planning] - Additional treatment: [if needed - further physio / injection / consideration for revision]

    Long-term Considerations: - Expected timeline to final outcome: [up to 12 months / 18 months] - Ongoing care: [monitoring for arthritis / hardware surveillance / other] - Activity modifications: [lifelong / temporary] - Footwear: [specific recommendations / orthotics]

    Common phrases: - Cleared for | may return to | can progress to - Continue | Stop | Maintain - Discharge to GP care | Open-door policy | Return PRN - If concerns arise | symptoms worsen | new problems develop - Annual review | Routine follow-up in [DURATION] - Expected full recovery by | Continue to improve over - Long-term: monitor for | surveillance of

    Example:

    Plan
- Activity progression:
  - Cleared for all activities including return to recreational basketball and hiking
  - No restrictions on activity level
  - Continue to build endurance and intensity gradually over the next 2-3 months

- Rehabilitation status:
  - Discharged from formal physiotherapy (completed 12-week program)
  - Continue home proprioception exercises for maintenance (balance board 3x/week)
  - ROM and strength goals achieved (equal to contralateral)

- Return to activities:
  - Work: Already returned full-time (desk job) - no issues
  - Driving: No restrictions
  - Sport: Begin gradual return to basketball - start with skills practice and light drills, progress to full play over 4-6 weeks as tolerated; cleared for hiking immediately

- Follow-up:
  - Discharge to GP care with open-door policy
  - Return PRN if concerns arise: recurrent instability, persistent pain, new symptoms
  - Routine follow-up not required unless problems develop

- Long-term considerations:
  - Small risk of recurrent instability (5-10%) - if occurs, return for assessment
  - Some residual stiffness compared to pre-injury normal is expected and should not limit function
  - No specific activity restrictions lifelong
  - Monitor for development of arthritis long-term (annual X-rays not needed unless symptomatic)

    3. Main Paragraph¶

    Format: Opening → Postoperative timeframe → Functional status review → Symptom status → Rehabilitation progress → Patient satisfaction → Examination (comparative to baseline and goals) → Imaging review with healing assessment

    Template:

    Functional Status: [PATIENT] reports [excellent / good / satisfactory / ongoing difficulty with] functional recovery. Currently able to: [specific functional milestones - walking unlimited / stairs without rail / returned to work / doing sport]. Compared to preoperative status: [much improved / significant improvement / some improvement / minimal change].

    Specific activities: - Walking: Distance [unlimited / limited to DISTANCE] [on flat ground / on uneven terrain / with aids] - Stairs: [managing independently / requires rail / difficulty] - Work: [returned full duties / modified duties / not yet returned] - Sport: [specific activities and level achieved]

    Symptom Status: Pain: [resolved / minimal / mild / moderate / severe]. Current level: [VAS or descriptive]. Location: [if persistent - where]. Character: [if persistent - description]. Timing: [at rest / with activity / after activity]. Compared to preoperative: [much better / improved / unchanged].

    Swelling: [resolved / minimal after activity / moderate / significant]. Timing: [constant / end of day / after specific activities]. Impact: [none / limiting footwear / limiting activity].

    Stiffness: [none / minimal / significant]. Timing: [morning / after rest / constant]. Impact: [none / limiting function / requires warmup].

    Instability or other symptoms: [describe if present / none reported].

    Rehabilitation Progress: Physiotherapy: [Completed NUMBER sessions / ongoing]. Focus: [ROM / strengthening / proprioception / sport-specific]. Achievements: [specific milestones reached]. Home program: [compliant / partially compliant / issues with].

    Progress in rehabilitation: [excellent / good / slower than expected / complications or setbacks].

    Patient Satisfaction: [PATIENT] reports being [very satisfied / satisfied / somewhat satisfied / dissatisfied] with the outcome. [Achieving / Has achieved / Working toward / Not yet achieved] preoperative goals of [specific goals]. Overall quality of life: [significantly improved / improved / unchanged / worse].

    Examination (Comparative):

    Inspection: - Scar(s): [well-healed / hypertrophic / keloid / other] - Swelling: [absent / minimal / moderate / significant] - Alignment: [maintained / appropriate / concerns] - Muscle atrophy: [resolved / minimal / still present]

    Palpation: - Tenderness: [resolved / minimal over LOCATION / significant over LOCATION] - Hardware: [not prominent / palpable / prominent / painful - if applicable] - Surgical area: [no tenderness / residual sensitivity]

    Range of Motion: - [JOINT]: Active [measurements] / Passive [measurements] - Compared to contralateral: [equal / 90% / 75% / 50% / <50%] - Compared to preoperative: [much improved / improved / similar / worse] - End feel: [normal / stiff / painful / soft tissue restriction] - Specific movements: [note any particularly limited or problematic movements]

    Strength: - [Muscle groups tested]: [5/5 / 4+/5 / 4/5 / 3/5 - compared to contralateral] - Compared to preoperative: [improved / similar / weaker] - Functional strength: [good / adequate / needs further work]

    Stability (if relevant): - [Ligament tests / stress tests]: [negative / stable / mild laxity / concerns] - Compared to preoperative: [much improved / improved / unchanged] - Patient perception: [feels stable / some instability / significant instability]

    Functional Tests: - Single leg stance: [able for SECONDS / unable] - Heel/toe walking: [able / unable / difficulty] - Squat/lunge: [able / unable / pain with] - Hop test: [able / unable / reluctant] (if appropriate for stage and procedure) - [Other relevant functional tests specific to procedure or goals]

    Follow-up Imaging Review:

    X-ray ([DATE], [views]):

    Hardware Assessment (if applicable): - Position: [stable / unchanged from previous / migrated / concerning changes] - All components: [intact / broken / loosened / backing out] - Compare to [DATE - previous X-ray]: [no change / progression noted]

    Alignment: - [Measurements - angles, distances]: [maintained / within normal limits / concerns] - Compare to immediate post-op: [stable / some loss of correction / significant change] - Compare to [DATE]: [stable / progressive change] - Overall alignment: [satisfactory / excellent / concerns]

    Bone Healing: - Fracture/osteotomy site: [healed / progressive healing / delayed union / nonunion] - Callus formation: [mature / bridging / hypertrophic / absent] - Bridging: [complete - 4/4 cortices / partial - 2-3/4 cortices / minimal - 0-1/4 cortices] - Time since surgery: [WEEKS/MONTHS] - [appropriate for timeline / delayed / concerning] - Quality of healing: [excellent / satisfactory / poor]

    For Arthrodesis (Fusion) Cases: - Joint space: [obliterated / narrowing / persistent gap] - Bone bridging: [complete across joint / partial / minimal / absent] - Fusion status: [solid / progressing well / questionable / nonunion] - Estimated consolidation: [~X% fused based on appearance] - Grade: [by grading system if used - Suckel fusion grades or similar]

    Complications: - Hardware failure: [none / specific issue] - Loss of reduction/alignment: [no / yes - describe] - Heterotopic ossification: [absent / present but not limiting / concerning] - Bone resorption or osteolysis: [none / present at LOCATION] - Arthritic changes in adjacent joints: [none / developing / significant] - Other: [any additional findings]

    Radiographic Healing Timeline Assessment: At [X weeks/months] post-operatively, radiographic findings demonstrate [excellent / satisfactory / appropriate / delayed / concerning] healing for this timeframe. [Specific assessment of progress]. [Expected to progress to complete healing / union within expected timeframe / concerning for delayed union or nonunion - may require extended monitoring or intervention].

    For Arthroplasty Cases: - Prosthesis position: [stable / no migration / subsidence noted] - Component alignment: [satisfactory / concerns] - Bone-prosthesis interface: [good integration / lucencies / loosening] - Wear: [not evident / minimal / concerning] - Compare to prior: [stable / changes noted]

    Common phrases: - returns for review | [TIME] following [PROCEDURE] - reports excellent functional recovery | is doing well - Currently able to [specific activities] - Walking unlimited distances | returned to work full-time | playing sport - Pain is [resolved / minimal] - Swelling [resolved / minimal after prolonged activity] - Completed [NUMBER] sessions of physiotherapy - Very satisfied with outcome | achieving goals - On examination, gait is normal - Scar well-healed | no tenderness | no swelling - Range of motion [equal to contralateral / 90% of contralateral] - Strength 5/5 throughout | improved from preoperative - Single leg stance able for [TIME] | heel/toe walking able - X-rays demonstrate | show | confirm - [Fracture/osteotomy/fusion] is [healed / progressing well / solid] - Hardware in situ, position stable - Bone bridging complete | callus formation excellent - At [TIME] post-op, findings show [appropriate / excellent] healing

    Example:

    Main Paragraph

[PATIENT] returns for review, 6 months following right ankle arthroscopy and lateral ligament reconstruction (Broström-Gould procedure with suture tape augmentation), performed on 15/07/25.

[PATIENT] reports excellent functional recovery and achievement of all preoperative goals. Currently able to walk unlimited distances without pain or instability, including on uneven terrain and hiking trails. Able to manage stairs without difficulty or need for handrail. Returned to full-time work (teaching, requires prolonged standing and walking) 3 months ago without issues. Recently resumed playing recreational basketball - has played 3 full games over the past month with no giving-way episodes, good confidence in the ankle, and minimal post-activity soreness that resolves overnight. Compared to preoperative status: dramatic improvement - was having 2-3 giving-way episodes per week and was unable to play any sport; now fully active without instability.

Pain: Essentially resolved. Occasionally (once every 1-2 weeks) experiences mild aching (1-2/10) in the ankle after very long or intense activity days, which resolves with rest overnight. No pain at rest, no pain with normal daily activities. Vastly improved from preoperative constant achiness (4-5/10) and severe pain (8/10) with giving-way episodes.

Swelling: Resolved. No residual swelling even after full day of activity or after basketball games. Initially had mild end-of-day swelling for the first 3-4 months but this has completely resolved.

Stiffness: Minimal. Slight stiffness for the first 5 minutes after waking in the morning, resolves quickly with walking. No stiffness with activities. Doesn't feel this limits function.

Instability: Completely resolved. No subjective sense of instability. No giving-way episodes since surgery. Confident on all surfaces including uneven ground. Able to jump and land without fear or sensation of ankle instability. Subjectively feels the ankle is "much more solid" than before surgery.

Rehabilitation progress: Completed 12-week formal physiotherapy program (April-June 2025) with excellent progress. Focused on ROM restoration (weeks 6-8), progressive strengthening (weeks 8-12), proprioception training (weeks 8-12), and sport-specific training (weeks 10-12). Home program compliance has been good - continued balance board exercises 3-4 times per week for maintenance. Formal therapy discharged 3 months ago.

Patient satisfaction: [PATIENT] is very satisfied with the surgical outcome. States "I'm so happy I had this done - I can do everything I want to do now without worry about my ankle giving way." Has achieved preoperative goals of: (1) return to basketball - achieved, (2) hiking without instability - achieved, (3) confident in ankle stability for daily activities - achieved. Quality of life significantly improved, no longer limited by ankle symptoms.

On examination:

Gait: Normal. No antalgic component. Normal heel strike, midstance, and toe-off phases. No circumduction. Preoperatively had cautious gait with shortened stance phase on right.

Inspection:
- Surgical scars (anterolateral and lateral ankle): Well-healed, thin, non-tender. Approximately 3cm and 5cm in length. No hypertrophy or keloid. Normal pigmentation.
- Swelling: Absent. Ankle contours symmetric to contralateral side.
- Alignment: Neutral hindfoot alignment maintained. No varus or valgus deformity.
- Muscle: No visible atrophy of peroneals or calf compared to left (had mild atrophy early postoperative period, now resolved).

Palpation:
- No tenderness over surgical sites
- No tenderness over lateral ligament complex
- No tenderness over peroneal tendons
- Anterolateral ankle: No tenderness or thickening
- Hardware not palpable (suture tape is internal)

Range of motion:
- Right ankle: Active dorsiflexion 12° / plantarflexion 48°
- Left ankle (unaffected): Active dorsiflexion 15° / plantarflexion 50°
- Right ankle: 96% of left ankle ROM (excellent)
- Preoperative right ankle: Dorsiflexion 15° / plantarflexion 50° (full ROM preoperatively; current slight restriction is common after ligament reconstruction due to intentional tightening)
- End feel: Firm capsular end feel, not painful
- Subtalar motion: Full inversion and eversion bilaterally, symmetric
- Pain: No pain throughout range of motion

Strength:
- Ankle dorsiflexion: 5/5 bilaterally (tibialis anterior)
- Ankle plantarflexion: 5/5 bilaterally (gastrocnemius-soleus)
- Ankle inversion: 5/5 bilaterally (tibialis posterior and invertors)
- Ankle eversion: 5/5 bilaterally (peroneals) - had been 4/5 at 3-month mark, now fully recovered
- Great toe extension and flexion: 5/5 bilaterally
- Compared to preoperative: Eversion strength improved (was 4/5 preoperatively due to disuse and deconditioning)

Stability:
- Anterior drawer test: Negative. ~3mm anterior translation (normal), compared to left ~3mm. Firm endpoint. Preoperatively: 10mm with soft/absent endpoint.
- Talar tilt test: Negative. ~5° tilt, compared to left ~5°. Preoperatively: 15° with marked apprehension.
- Subjectively: Ankle feels very stable to patient. No apprehension with stress testing.
- Overall: Excellent stability restored

Functional tests:
- Single leg stance: Able to maintain >30 seconds eyes open on right, >30 seconds eyes closed. Equal to left. Excellent proprioception recovered.
- Heel walking: Able without difficulty
- Toe walking: Able without difficulty
- Single leg heel raise: Able x 20 repetitions on right (equal to left x 20)
- Squat: Able to full depth without pain or instability
- Hop test: Able to perform single-leg hops on right with good control, no pain, no subjective instability. Not formally measured but appears equal to contralateral.

Follow-up imaging:

X-ray right ankle (today's visit, weight-bearing AP, lateral, and mortise views):

Hardware assessment:
- Suture tape augmentation (internal brace) present, running from fibula to talus
- Position: Stable, unchanged from 3-month postoperative X-rays
- Fibular anchor site: Well-incorporated, no lucency or bone resorption
- Talar anchor site: Well-incorporated, no lucency
- No hardware failure, no migration

Alignment:
- Tibiotalar alignment: Neutral (satisfactory)
- Hindfoot alignment: Neutral on mortise and lateral views
- No varus or valgus tilt
- Talar dome position within mortise: Centered
- Compare to immediate post-op: Unchanged, alignment maintained

Bone:
- No avascular necrosis of talus
- No fractures
- Bone density normal
- Anchor sites healed without complication

Joint:
- Tibiotalar joint space preserved (3-4mm, symmetric to contralateral)
- No evidence of post-traumatic arthritis
- No osteophytes
- Cartilage space maintained (excellent - preserved cartilage preoperatively is holding up well)

Soft tissues:
- No significant effusion
- No calcifications or heterotopic ossification
- Soft tissue contours normal

Complications:
- No hardware failure or migration
- No loss of alignment
- No development of arthritis (favorable at 6 months)
- No bone resorption at anchor sites
- No other complications visualized

Radiographic assessment:
At 6 months post-operative, X-rays demonstrate maintained alignment with stable hardware position and excellent preservation of joint space. The suture tape internal brace is in satisfactory position and well-incorporated. Most importantly, there is no evidence of post-traumatic arthritis development, which is favorable (one of the goals of ligament reconstruction is to restore stability to prevent cartilage damage and arthritis development). The bone anchor sites are fully healed without lucency or resorption. Overall radiographic appearance is excellent for this timeframe and supports the clinical finding of successful stabilization.

    4. Impression and Recommendation¶

    Format: Summary of recovery quality → Compare to goals → Imaging interpretation → Activity clearance and guidance → Discharge vs follow-up → Long-term monitoring

    Template:

    Recovery Summary: In summary, [PATIENT] has achieved [excellent / good / satisfactory / suboptimal] functional outcome following [PROCEDURE], now [TIME] post-operatively. [Overall assessment of recovery trajectory and quality].

    Goal Achievement: [PATIENT] [has achieved / is achieving / is progressing toward / has not yet achieved] the preoperative goals of [specific goals]. [Comparison to baseline / preoperative status]. [Patient satisfaction level].

    Radiographic Assessment: Radiographic assessment demonstrates [solid fusion / excellent healing / stable hardware / preserved joint / other relevant findings]. [Specific positive findings]. [Any concerns or findings requiring monitoring].

    Functional Status: Functionally, [PATIENT] [has returned to all desired activities / is working toward return / has residual limitations of SPECIFIC]. [Specific activity clearances or restrictions].

    Activity Clearance and Guidance:

    I have cleared [PATIENT] for: - [Specific activities with any qualifications or progression guidance] - [Work status - full duties / sport clearance / driving / etc.] - [Any ongoing restrictions if applicable, with duration]

    [Specific advice for progression - gradual increase / sport-specific guidance / listen to body / reasonable activity modifications].

    Rehabilitation and Home Program: - [Continue / discharge from] formal physiotherapy - [Continue / no longer needed] home exercise program with [specific guidance] - [Maintenance exercises] recommended: [specific exercises and frequency]

    Discharge or Continued Follow-up:

    If Excellent Progress / Discharge: Given the excellent functional outcome and achievement of all goals, I am discharging [PATIENT] to GP care with an open-door policy. [PATIENT] should return if: - [Specific reasons to return - symptoms worsen / instability recurs / new problems / concerns] - [Routine follow-up in TIMEFRAME if indicated - e.g., annual for arthroplasty, hardware removal consideration]

    If Ongoing Monitoring Needed: I will continue to follow [PATIENT] with review in [TIMEFRAME] to: - [Specific goals for next visit] - [Assessment needed / decision pending]

    Long-term Considerations and Counseling: - Expected final outcome: [Continue to improve over DURATION / current status likely final outcome] - Risk of [recurrence / progression / adjacent joint issues]: [Specific risk and counseling] - Activity recommendations long-term: [Any modifications or avoidances / no restrictions] - Monitoring: [What to watch for - specific symptoms / when to seek review] - Hardware considerations: [Removal planned/not planned / surveillance needed]

    Common phrases: - In summary | Overall - has achieved excellent functional outcome - following [PROCEDURE], now [TIME] post-operatively - has achieved all preoperative goals - is very satisfied with the outcome - Radiographic assessment demonstrates | confirms | shows - solid fusion | excellent healing | stable hardware | preserved joint - Functionally | From a functional standpoint - has returned to | is able to perform | has achieved - I have cleared [PATIENT] for - all activities | return to sport | full duties - no restrictions | avoid [ACTIVITY] for [DURATION] - I am discharging to GP care | discharge with open-door policy - Return if symptoms worsen | concerns arise | new problems - I will continue to follow | review in [TIMEFRAME] - Expected to continue improving | final outcome assessment - Risk of [COMPLICATION] | Monitor for | Watch for - Thank you for allowing me to participate in [PATIENT]'s care

    Example:

    Impression and Recommendation

In summary, [PATIENT] has achieved an excellent functional outcome following right ankle arthroscopy and lateral ligament reconstruction (Broström-Gould with suture tape augmentation), now 6 months post-operatively. The recovery trajectory has been exemplary with steady progression through rehabilitation milestones and achievement of all functional goals well within the expected timeframe.

[PATIENT] has achieved all preoperative goals: (1) restoration of ankle stability without giving-way episodes - achieved with no episodes of instability in 6 months and excellent stability on examination; (2) return to recreational basketball - achieved, now playing regularly without limitations; (3) hiking on uneven terrain without fear - achieved without issues; and (4) pain-free daily activities - achieved with essentially no pain. Compared to the preoperative status of chronic instability with 2-3 giving-way episodes per week, persistent pain, and inability to participate in any sport, the improvement has been dramatic and life-changing for the patient. [PATIENT] is very satisfied with the surgical outcome.

Radiographic assessment at 6 months demonstrates maintained anatomic alignment with stable hardware position (suture tape internal brace well-incorporated) and, most importantly, preserved tibiotalar joint space without evidence of post-traumatic arthritis. This is an excellent finding, as one of the primary goals of surgical stabilization is to restore stability to prevent ongoing cartilage damage and subsequent arthritis that can occur with recurrent instability. The bone anchor sites are fully healed. Overall radiographic appearance supports the clinical finding of successful stabilization and is very favorable for long-term outcome.

Functionally, [PATIENT] has returned to all desired activities without restriction. Range of motion is 96% of the contralateral ankle (slight restriction is common and expected after ligament reconstruction due to intentional ligamentous tightening for stability). Strength has fully recovered to 5/5 in all muscle groups including the peroneals, which had been weak preoperatively. Proprioception has fully recovered with excellent single-leg stance performance. Most importantly, stability has been completely restored with negative stress tests and subjective confidence in the ankle across all activities including high-demand sport.

I have cleared [PATIENT] for:
- All activities including recreational basketball, hiking, and any other desired sports - no restrictions on activity level
- Full return to work without modifications (already achieved)
- No restrictions on footwear, though supportive athletic shoes recommended for high-impact activities (standard recommendation, not a restriction due to surgery)
- Gradual progression over the next 2-3 months to build sport-specific endurance and confidence, but from a healing and stability standpoint, the ankle is ready for all activities

Rehabilitation and home program:
- Formal physiotherapy has been completed and [PATIENT] discharged from physical therapy
- Home exercise program can be transitioned to maintenance level: Continue balance board exercises 2-3 times per week for proprioception maintenance (especially important for cutting/pivoting sports like basketball to reduce risk of any recurrent injury). This can be integrated into warm-up routine for sports.
- No other specific exercises required; general fitness and sport-specific training sufficient at this point

Given the excellent functional outcome, achievement of all goals, and complete resolution of the preoperative instability problem, I am discharging [PATIENT] to GP care with an open-door policy. [PATIENT] should return to see me if:
- Symptoms worsen: return of instability, giving-way episodes, persistent pain
- Trauma: new significant ankle injury requiring assessment
- Concerns: any concerns about the ankle or surgical outcome
- Routine follow-up is not required unless any of the above issues develop

Long-term considerations and counseling:

Expected final outcome: [PATIENT] is essentially at the final outcome now at 6 months. May continue to see very minor improvements in ROM and any residual stiffness over the next 6 months (up to 12 months post-op for absolute final outcome), but current status is excellent and represents successful surgery.

Risk of recurrent instability: The recurrence rate after Broström-Gould with suture tape augmentation is low (approximately 5% in the literature). The excellent stability on examination, good surgical tissue quality noted intraoperatively, use of augmentation for additional security, and patient's excellent compliance with rehabilitation all suggest a low personal risk of recurrence. That said, [PATIENT] has been counseled about the signs of recurrent instability (return of giving-way, sense of looseness) and knows to return if this develops. To minimize risk: continue proprioception exercises as part of warm-up routine, wear appropriate supportive footwear for sport, and use ankle brace for very high-risk activities if desired (though not required).

Risk of post-traumatic arthritis: The surgery was performed in part to prevent development of arthritis by restoring stability (unstable ankles are at high risk of progressive cartilage damage). The 6-month X-rays show preserved joint space which is very favorable. Long-term, there remains some risk of arthritis development over years/decades (any injury to the ankle increases lifetime arthritis risk), but this surgery significantly reduces that risk by restoring normal biomechanics. No routine X-ray surveillance is needed unless symptoms develop.

Activity recommendations long-term: No specific restrictions. [PATIENT] can participate in any desired activities. Basketball and other cutting/pivoting sports are allowed. Should wear appropriate supportive athletic shoes for high-impact sports (good practice regardless). May use prophylactic ankle brace for basketball if desired for psychological comfort, though not required from a medical standpoint given the excellent stability. Should continue to maintain good overall lower extremity strength and flexibility as part of injury prevention.

Hardware considerations: The suture tape augmentation does not require removal and will remain in situ permanently. It is well-incorporated and not causing any symptoms. No planned hardware removal. If [PATIENT] were to develop pain or prominence over the hardware sites in the future (uncommon), could consider removal at that time, but currently no indication.

What to monitor for / When to return:
- Watch for return of instability symptoms (giving-way, sense of looseness) - if occurs, return for assessment promptly
- Watch for development of persistent ankle pain, swelling, or stiffness that doesn't resolve - could indicate arthritis development or other issue - return for assessment and X-rays
- If sustains significant new ankle injury (trauma), seek assessment to ensure no new damage to ligaments or other structures
- Otherwise, no routine follow-up needed - discharged with open-door policy

Thank you for allowing me to participate in [PATIENT]'s care. It has been a pleasure to achieve such an excellent outcome. Please do not hesitate to contact me if any concerns arise in the future.

Yours sincerely,

Dr. [YOUR NAME]
Orthopaedic Surgeon

Main Paragraph

[PATIENT] returns 6 months following right ankle arthroscopy and lateral ligament reconstruction. Reports excellent functional recovery: walking unlimited, returned to work, playing basketball without instability. Pain resolved. Swelling resolved. No giving-way episodes. Completed 12-week physiotherapy. Very satisfied with outcome. Achieved all preoperative goals.

On examination: Gait normal. Scars well-healed. No tenderness. No swelling. ROM: Right ankle DF 12°/PF 48° (96% of contralateral). Strength 5/5 throughout. Anterior drawer and talar tilt tests negative. Single leg stance >30 seconds. Excellent proprioception.

X-rays: Suture tape augmentation in stable position, well-incorporated. Alignment maintained. Joint space preserved, no arthritis. Anchor sites healed.

Impression and Recommendation

In summary, [PATIENT] has achieved excellent functional outcome following right ankle arthroscopy and lateral ligament reconstruction, now 6 months post-operatively.
